## Creating Your Wallet

### Step-by-Step Setup

<Steps>
  <Step title="Launch M.O.N.K.Y">
    Open your web browser and navigate to the M.O.N.K.Y wallet interface
  </Step>
  
  <Step title="Create New Wallet">
    Click "Create New Wallet" to begin the setup process
  </Step>
  
  <Step title="Generate Seed Phrase">
    Your unique 12-word seed phrase will be automatically generated
  </Step>
  
  <Step title="Secure Your Seed Phrase">
    Write down your seed phrase and store it in a safe location
  </Step>
  
  <Step title="Verify Seed Phrase">
    Confirm your seed phrase by selecting words in the correct order
  </Step>
  
  <Step title="Set Password">
    Create a strong password to protect your wallet locally
  </Step>
  
  <Step title="Complete Setup">
    Your M.O.N.K.Y wallet is now ready to use!
  </Step>
</Steps>

### Import Existing Wallet

Already have a Solana wallet? Import it easily:

<AccordionGroup>
  <Accordion title="From Seed Phrase">
    **12 or 24 Word Recovery**
    
    1. Select "Import Wallet" option
    2. Enter your existing seed phrase
    3. Set a new password for M.O.N.K.Y
    4. Confirm and complete import
  </Accordion>

  <Accordion title="From Private Key">
    **Direct Key Import**
    
    1. Choose "Import from Private Key"
    2. Enter your base58 encoded private key
    3. Set wallet password
    4. Verify and finish setup
  </Accordion>

  <Accordion title="From Hardware Wallet">
    **Ledger/Trezor Integration**
    
    1. Connect your hardware wallet
    2. Select "Hardware Wallet" option
    3. Choose your device type
    4. Follow on-screen instructions
  </Accordion>
</AccordionGroup>

## Security Setup

### Essential Security Measures

<CardGroup cols={2}>
  <Card title="🔐 Strong Password" icon="lock">
    **Password Requirements**
    
    • Minimum 12 characters
    • Mix of letters, numbers, symbols
    • No common dictionary words
    • Unique to your wallet
  </Card>
  
  <Card title="📝 Seed Phrase Backup" icon="file-text">
    **Backup Best Practices**
    
    • Write on paper, never digital
    • Store in multiple secure locations
    • Consider fireproof/waterproof storage
    • Never share with anyone
  </Card>
  
  <Card title="📱 2FA Setup" icon="shield">
    **Two-Factor Authentication**
    
    • Enable SMS or app-based 2FA
    • Use authenticator apps like Google Authenticator
    • Backup your 2FA recovery codes
    • Required for large transactions
  </Card>
  
  <Card title="🔍 Phishing Protection" icon="eye">
    **Stay Safe Online**
    
    • Bookmark official M.O.N.K.Y URL
    • Verify SSL certificate
    • Never enter seed phrase online
    • Check for official social media accounts
  </Card>
</CardGroup>

## Troubleshooting Setup

### Common Issues

<AccordionGroup>
  <Accordion title="Seed Phrase Not Working">
    **Import Issues**
    
    • Check for typos in seed words
    • Ensure correct word order
    • Verify derivation path settings
    • Try different import methods
    • Contact support if persistent
  </Accordion>

  <Accordion title="Connection Problems">
    **Network Issues**
    
    • Check internet connection
    • Try different RPC endpoints
    • Disable VPN temporarily
    • Clear browser cache
    • Update browser to latest version
  </Accordion>

  <Accordion title="Balance Not Showing">
    **Sync Issues**
    
    • Wait for network synchronization
    • Refresh the wallet interface
    • Check transaction on Solana explorer
    • Verify correct network selection
    • Try reconnecting wallet
  </Accordion>
</AccordionGroup>
