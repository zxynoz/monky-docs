## Security Questions

<AccordionGroup>
  <Accordion title="How secure is M.O.N.K.Y?">
    M.O.N.K.Y uses military-grade security measures:
    - **256-bit encryption** for all sensitive data
    - **Local key storage** - your keys never leave your device
    - **Hardware wallet support** for additional security
    - **Multi-factor authentication** options
    - **Regular security audits** by third-party firms
  </Accordion>

  <Accordion title="Can M.O.N.K.Y access my private keys?">
    No, absolutely not. M.O.N.K.Y operates on a zero-knowledge architecture. Your private keys are generated and stored exclusively on your device. We cannot access your funds even if we wanted to.
  </Accordion>

  <Accordion title="What should I do if I lose my recovery phrase?">
    Unfortunately, if you lose your recovery phrase and don't have access to your wallet, there's no way to recover your funds. This is by design for security reasons. Always:
    - Store your recovery phrase in multiple secure locations
    - Never store it digitally
    - Consider using a hardware wallet for large amounts
  </Accordion>

  <Accordion title="Is my data safe with M.O.N.K.Y?">
    Yes. We follow strict privacy principles:
    - No personal data collection beyond what's necessary
    - All communications are encrypted
    - We never sell or share your data
    - GDPR and CCPA compliant
  </Accordion>
</AccordionGroup>

## Troubleshooting

<AccordionGroup>
  <Accordion title="My transaction is stuck or failed. What do I do?">
    If your transaction is stuck:
    1. **Wait**: Solana transactions usually confirm in 1-2 seconds, but network congestion can cause delays
    2. **Check status**: Use a Solana explorer to check your transaction status
    3. **Retry**: If failed, you can retry the transaction
    4. **Contact support**: If issues persist, reach out to our support team
  </Accordion>

  <Accordion title="I can't connect my hardware wallet">
    Try these steps:
    1. **Browser compatibility**: Ensure you're using a supported browser (Chrome recommended)
    2. **Enable WebUSB**: Make sure WebUSB is enabled in your browser
    3. **Firmware update**: Ensure your hardware wallet has the latest firmware
    4. **Restart**: Try restarting your browser and hardware wallet
    5. **Contact support**: If issues persist, contact our support team
  </Accordion>

  <Accordion title="The wallet is loading slowly">
    To improve performance:
    1. **Clear cache**: Clear your browser cache and refresh
    2. **Disable extensions**: Temporarily disable browser extensions
    3. **Check connection**: Ensure you have a stable internet connection
    4. **Update browser**: Make sure you're using the latest browser version
    5. **Try incognito**: Test in an incognito/private browsing window
  </Accordion>

  <Accordion title="I'm getting error messages">
    Common solutions:
    1. **Refresh page**: Try refreshing the M.O.N.K.Y app
    2. **Clear storage**: Clear local storage and restart
    3. **Check network**: Verify Solana network status
    4. **Update browser**: Ensure browser is up to date
    5. **Screenshot**: Take a screenshot of the error and contact support
  </Accordion>
</AccordionGroup>
